# Smart City DBMS Project (SQL)

## Project Overview
This project demonstrates the design and implementation of a Smart City database using SQL. It covers database schema creation, data insertion, and complex SQL queries using joins and subqueries.

## Files Included
- 01_Database_Schema.sql – Database and table creation
- 02_Insert_Data.sql – Sample data insertion
- 03_SQL_Queries.sql – SQL queries with joins and subqueries

## Key Concepts Used
- Database normalization
- Primary and foreign keys
- JOIN operations
- Subqueries
- Data filtering and aggregation

## Tools Used
- MySQL
- GitHub
